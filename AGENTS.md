## Cursor Cloud specific instructions

This is a simple Node.js CLI application with zero npm dependencies. All available scripts are in `package.json`:

- `npm start` — runs `src/index.js` (prints a welcome message and exits)
- `npm run dev` — runs `src/index.js` with `--watch` (auto-restarts on file changes)
- `npm test` — placeholder that echoes "Sin pruebas configuradas" and exits 0

No database, Docker, or external services are required. The project has no linter configured and no build step.
